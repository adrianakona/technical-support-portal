# Technical Support & IT Asset Management Portal

## Project Overview: 
A centralized digital hub developed during my 2024 academic internship at FGCL (SSII).

## Purpose
During my internship, I identified a gap in how IT assets were tracked and how technical issues were reported. I built this portal using Google Sites to streamline the communication between staff and the IT department. It serves as a first-line support tool to reduce the time spent on repetitive troubleshooting tasks.

## Features
- Troubleshooting Guides: Step-by-step documentation for common hardware and network issues.
- Knowledge Base: Centralized technical manuals and FAQ for company software.

## Impact
The site was designed to empower non-technical staff to solve minor issues independently, allowing the IT team to focus on more complex network security tasks. It demonstrated how a "no-code" solution can be rapidly deployed to solve real-world organizational bottlenecks.
